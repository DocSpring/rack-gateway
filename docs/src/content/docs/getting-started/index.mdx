## Who Should Use Rack Gateway?

Rack Gateway is designed for teams who:

- **Self-host Convox racks** and need enterprise-grade access controls
- **Require SOC 2 compliance** for their infrastructure
- **Want individual accountability** instead of shared credentials
- **Need audit trails** for infrastructure access
- **Prefer open source** solutions they can customize and self-host
