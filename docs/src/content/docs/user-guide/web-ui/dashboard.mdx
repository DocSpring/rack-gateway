The **Rack** page is the landing view of the web UI. It summarizes the connected Convox rack and its current status.

## What You See

The rack summary includes:

- **Name**
- **Domain**
- **Rack Domain**
- **Provider**
- **Region**
- **Type**
- **Version**
- **Count** (cluster size)
- **Status**

If the gateway cannot load rack metadata, the page shows an error banner with the failure reason.

<Aside type="note">
There is no separate “dashboard” screen. The Rack page is the default overview.
</Aside>
