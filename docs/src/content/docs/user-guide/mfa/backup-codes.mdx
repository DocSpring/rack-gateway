Backup codes provide emergency access if you lose your primary MFA device.

## Key Facts

- 10 one-time codes generated per batch
- Each code works once
- Codes are only shown when generated

<Aside type="danger">
Store backup codes securely. If you lose them, recovery may require admin assistance.
</Aside>

## Getting Backup Codes

### During Enrollment

When you enable MFA, the UI displays a fresh set of backup codes. Download or copy them immediately.

### Regenerating Codes

<Steps>

1. Go to **Account Security**
2. Open **Backup Codes**
3. Click **Regenerate backup codes**
4. Download the latest codes

</Steps>

Regenerating codes invalidates all previous codes.

## Using a Backup Code

### Web UI

Enter a backup code when prompted for MFA.

### CLI

```bash
rack-gateway login
```

When prompted, paste a backup code instead of a TOTP code.

## Best Practices

- Store codes in a password manager or secure vault
- Regenerate after heavy use
- Treat codes like passwords
